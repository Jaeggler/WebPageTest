Design Network

I will use a Arduino One board to send serial data to an output board.

Simulation

First I will simulate a serial comm network. Using two Arduino Boards, I will send data (1 and 2), and the other board will light one LED each time.

I'm using Rx and Tx pin (RS232) for this test.

The program used for this test is at the bottom of the page.

With self board

Using Neil RS232 programming, I set the Arduino One to send pulse equal to the output board Node code (Check code at the files). When reciving data, the LED on the output board blinks slowly. When no data is coming blinks faster.

All programming its done with a Arduino ONE as an ISP.
