Micro Datasheet

In this assigment I will ve using the Electronics Design week board I did. This is an attiny45 micro from AVR. I found the micro datasheet at this link.

Reading a micro might be a long and messy task. I used Sparkfun Electronics guide "How to read a Datasheet" (link here), which summaries the most important stuff when not having experience with electronics.

According to the guide, I will be looking for: summary, specifications, pinout, absolute ratings, recommended operation conditions and example schematics.

Attiny 45 it's a 8 pin microcontroller. Taking VCC and GROUND pins, the micro have 6 I/O pins from which usually you dont use the RESET one. The micro have 4K storage, meaning it have really short space. The lighter the code, the better.

Programming

I will be using the Arduino IDE to check memory difference between Arduino Languaje and C Languaje. I will download a blink code with Arduino and C Languaje to compare main diferences.

Arduino languaje

Using Arduino IDE I programmed the blink code (files bottom of page). When loaded to the W6 board I can see it uses 22% of the storage. Datasheet details the program memory size is 4KB.

C Languaje

Using Arduino IDE I programmed the blink code with C code (files bottom of page). When loaded to the W6 board I can see it uses only 2% of the storage. Meaning everytime I program something , I'm adding all library to the memory. Bad side of this, its C code it's really unfrindly making it better to situation where you need to improve your storage. For quick works, using Arduino would be better.
